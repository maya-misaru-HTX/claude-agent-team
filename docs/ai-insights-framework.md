# [Company] AI Insights Framework

The core philosophy for all consumer insight reports and research output.

## The One Rule

**Your job is not to summarize the data. Your job is to explain what the data means for the brand.**

## 10 Principles

### 1. Insights Are Explanations, Not Data
Every statistic must answer one of three questions:
- Why did this happen?
- What does it reveal about the consumer?
- What should the brand do because of it?

If none are answered, the statistic should not appear.

### 2. Slides Connect to a Narrative
The deck should feel like one argument unfolding, not a collection of charts. Structure: Context -> Big Consumer Truth -> Proof -> Nuance -> Implications. Every slide must answer: "How does this slide advance the story?"

### 3. Start With the Answer, Not the Data
Headlines are conclusions, not topics.
- Bad: "Product rankings"
- Good: "Classic silhouettes win because they signal authentic Western identity"

### 4. Numbers Only Matter When They Change Decisions
Always ask: "If the brand did not know this statistic, would it change their decision?" If no, remove it.

### 5. Segmentation Reveals Strategy
Segment analysis must answer at least one:
- Where should the brand invest?
- Where should the brand avoid investing?
- Where should messaging change?

### 6. Consumers Care About Meaning, Not Features
Translate product features into consumer meaning. Material preference isn't "goat leather vs bison" -- it's authenticity, durability, craftsmanship, identity.

### 7. Identify Category Rules
The strongest insights identify what the category allows vs what the category rejects. Consumers accept innovation within category identity boundaries.

### 8. Create Strategic Tension
Always surface 2-3 tensions the brand must navigate: Authenticity vs Innovation, Volume vs Premium, Classic vs Fashion.

### 9. Good Next Steps Are Specific
Recommendations must include: Action + Rationale + Expected Impact.
- Weak: "Lean into exotics for premium segments."
- Strong: "Position exotic leathers as limited-run halo products that elevate brand prestige rather than expanding them as core assortment."

### 10. Write Like a Category Strategist
- Bad: "The data suggests consumers prefer goat leather."
- Good: "Goat leather emerges as the category sweet spot -- durable enough for everyday wear but refined enough for social settings."
